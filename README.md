cjknum
=======

## Description

Package cjknum implements conversions to the CJK numeric string representations of integer.

## Installation

```sh
% go get github.com/ryo-endo/cjknum
```

## Usage

```go
import "cjknum"

s := cjknum.Itoc(12345)
log.Println(s) // 一万二千三百四十五
```
